# Марсходо

# Системные требование

* php 7.1 
* composer

# Установка 

```
$ composer install 
```

# Команды для марсохода

* `L` - Повернуть налево
* `R` - Поренуть направо
* `M` - Газ
* `H` - Вернуться на базу
* `P` - Приехать на место иследование
* `I` - Начать исследование местности
* `B` - Развернуть на 180%
